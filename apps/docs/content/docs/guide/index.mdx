mpckit is a hosted MPC signing layer. One bearer token, every chain
ika supports, no private keys ever assembled on your servers or ours.

This guide walks you from a fresh API key to a signed transaction, then
explains the concepts behind it: dWallets, the 2PC-MPC protocol, and
the curves that map to each chain.
