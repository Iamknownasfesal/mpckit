```ts
const list = await api.listDWallets();   // { dwallets: DWallet[] }
const detail = await api.getDWallet(id); // { dwallet: DWallet }
```

Both calls are simple HTTP reads against the backend's mirror of Sui
state.

## `DWallet`

  type={{

/>

The Sui record at `suiDwalletId` is the source of truth. The backend
row mirrors it for fast reads. See
[Concepts: dWallets](/docs/guide/dwallets) for the deeper mental
model.

## Filtering by status

```ts
const list = await api.listDWallets();
const ready = list.dwallets.filter((d) => d.status === "active");
```

## Bypassing the cache

If you need to read on-chain state directly:

```ts
const onchain = await api.raw.get(`/v1/dwallets/${id}/onchain`);
```

The escape hatch is documented in
[Utilities](/docs/typescript/utilities).
